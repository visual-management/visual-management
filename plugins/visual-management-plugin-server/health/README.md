---

## Configuration

### urls

`Array<string>`

URLs to watch.

### showWorkingUrls

`boolean`, defaults to `false`

If you should show (or not) the URLs in success.

### updateInterval

`number` (in milliseconds), defaults to `60000` (1 minute)

Update component info at this time interval.
